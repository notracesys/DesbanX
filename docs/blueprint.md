# **App Name**: FF Account Assist

## Core Features:

- Initial Account Quiz: A strategic quiz to gather information about the user's account and ban circumstances, influencing personalized messaging.
- Data Analysis Engine: Back-end data evaluation. The engine determines the likelihood of account recovery based on user inputs, cross-referencing data with successful appeal patterns. Generative AI tool will use its reasoning ability when determining what information should be considered when presenting its output.
- Personalized Result Display: Presents customized results based on quiz answers and the Data Analysis Engine, indicating the strength of recovery possibilities with a disclaimer about guaranteed recovery.
- Case Analysis: A technical breakdown of the user's ban, identifying the type and potential errors.
- Appeal Preparation: Crafting a strategic appeal message using appropriate language for official support, based on policies.
- Account ID Verification: A feature where users enter their Free Fire ID, and the system uses permitted APIs to display public data such as nickname, level, and server.
- Paywall Integration: Clear explanation that unlocking complete analysis and an optimized appeal requires service purchase.

## Style Guidelines:

- Primary color: Dark red (#8B0000) to resonate with Free Fire's branding while keeping a professional feel.
- Background color: Dark gray (#2F4F4F) to provide a contrasting backdrop for content.
- Accent color: Orange (#FF8C00) as a highlight for calls to action and important information, creating a sense of urgency and grabbing the user's attention.
- Font: 'Space Grotesk' (sans-serif) for headers paired with 'Inter' (sans-serif) for body text. 'Space Grotesk' lends a modern, techy edge suitable for headlines, while 'Inter' ensures readability and neutrality for longer texts.
- System analysis-style icons, avoiding any game logos, aiming for a clean and trustworthy aesthetic.
- Fluid, responsive design focusing on guiding users smoothly from the initial quiz to the checkout process.
- Subtle animations on the data processing screen and a celebratory confetti animation upon positive result display.
